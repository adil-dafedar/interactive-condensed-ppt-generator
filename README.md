# Interactive Condensed PPT Generator

A FastAPI app that generates concise slides using AI.

## Features
- ChatGPT-style UI
- Dark/light theme
- AI slide generation (Gemini)
- Optional images
- PDF download
- Keyboard navigation

## Tech Stack
- FastAPI
- HTML/CSS
- Gemini API

## Setup
1. Clone repo
2. Install requirements
3. Add .env
4. Run: uvicorn main:app --reload
